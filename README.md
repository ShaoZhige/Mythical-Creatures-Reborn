# Mythical Creatures Reborn（神话生物重制版）

将 MLP Mythical Creatures (1.7.10) 移植到 Minecraft 1.20.1 Forge并革新。

*A modernized 1.20.1 Forge port of the MLP Mythical Creatures mod with brand-new features.*

```bash
# JDK 17
./gradlew build
```

JAR 输出在 `build/libs/`。
