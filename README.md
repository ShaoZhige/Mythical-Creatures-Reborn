# Mythical Creatures Reborn

将 MLP Mythical Creatures (1.7.10) 移植到 Minecraft 1.20.1 Forge 并全面革新。

A modernized 1.20.1 Forge port of the MLP Mythical Creatures mod, featuring 159+ items, 17 blocks, 8 armor sets, 9 entities, and 102 recipes.

## 模组信息 / Info

| | |
|---|---|
| MC 版本 / Version | 1.20.1 |
| 加载器 / Loader | Forge 47.2.0+ |
| 许可 / License | MIT |
| 作者 / Authors | Shao_Zhige / fffqu / Holy Light Radiance |

## 内容概览 / Overview

- **武器** / Weapons — 13 把剑，多把带特殊技能
- **弓 & 投掷物** / Bows & Throwables — 紫悦之弓、凤羽弓、陨星火球、彩虹云等
- **工具** / Tools — 5 套完整工具 + 3×3×3 范围挖掘者
- **盔甲** / Armor — 8 套盔甲，多套含全套特效
- **可爱标志** / Cutie Marks — 6 个角色标志，永久 Buff，支持 Curios 饰品槽 + 腿部渲染
- **实体** / Entities — 紫悦（可驯服）、紫悦魔法、彩虹光柱等
- **效果** / Effects — 流血、修补
- **方块** / Blocks — 15 个建筑与材料方块

> 详细内容见 Wiki / See [Wiki](https://github.com/ShaoZhige/Mythical-Creatures-Reborn/wiki) for details.
