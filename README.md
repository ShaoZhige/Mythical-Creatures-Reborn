# Mythical Creatures Reborn（神话生物重制版）

将 MLP Mythical Creatures (1.7.10) 移植到 Minecraft 1.20.1 Forge并革新。

## 模组信息

| | |
|---|---|
| 版本 | 1.1.0-Forge-1.20.1 |
| 加载器 | Forge 47.2.0+ |
| 作者 | Shao_Zhige  / fffqu / Holy Light Radiance |
| 许可 | MIT |

## 内容

### 武器

| 武器 | 特殊技能 |
|---|---|
| 天角兽之剑 | 3D 模型 |
| 紫悦之剑 | 右键召唤紫悦魔法 / 潜行右键削减血量 1/6 |
| 云宝之剑 | 右键彩虹云 / 潜行右键持续光束 |
| 库巴之剑 | 右键发射陨星火球（TNT 级爆炸 + 黑色粒子） |
| 随机爆炸 | 右键方块变为岩浆源 |
| 熊爪剑 | 攻击施加**流血** debuff（移动翻倍伤害） |
| 苹果剑 | — |
| 水晶剑 | — |
| 苹果嘉儿剑 | — |
| 匕首 | — |
| 泥土剑 | 很便宜算不算？ |

### 弓

| 弓 | 弹药 |
|---|---|
| 紫悦之弓 | 紫悦之星（创造模式无需弹药） |
| 凤羽弓 | 烈焰粉（无限附魔不消耗） |

### 工具

库巴、紫悦、苹果、熊爪、苹果嘉儿系列（各含镐/斧/锄/锹）

| 特殊工具 | 效果 |
|---|---|
| 挖掘者 | 钻石挖掘等级 + 铁速度 + **3×3×3 范围挖掘** |
| 库巴之杆 | 右键方块变为岩浆源 |
| 库巴之眼 | 右键召唤闪电 |

### 投掷物品

| 物品 | 效果 |
|---|---|
| 陨星火球 | 爆炸半径 5.0（>TNT），伤害 13，着火 5 秒，黑色烟雾粒子 |
| 凤凰羽毛 | 直射 + 命中点燃 5 秒 |
| 紫悦之星 | 命中召唤闪电 + 紫色粒子 |
| 彩虹云 | 爆炸 + 白色云朵粒子 |
| 苹果 | 可投掷 |

### 盔甲套装

库巴、紫悦、黑晶、苹果、熊皮、云宝、苹果嘉儿

- 云宝套装：全套创造飞行 + 免疫摔落伤害
- 苹果嘉儿套装：全套伤害吸收 I
- 库巴套装：全套 +15 最大生命 + 防火
- 熊皮套装：全套 +4 攻击伤害
- 紫悦套装：全套 +20 最大生命

### 实体

| 实体 | 描述 |
|---|---|
| 紫悦（Twilight Sparkle） | 友好生物 |
| 紫悦的魔法 | 召唤物，追踪敌人造成魔法伤害 |
| 彩虹光柱 | 持续光束，旋转彩虹纹理 |

### 效果

| 效果 | 描述 |
|---|---|
| 流血 | 每秒掉血，移动伤害翻倍 |
| 修补 | 每 3 秒全背包 + 装备 + 主副手回复 1 点耐久 |

### 可爱标志

放入背包或 Curios 可爱标志栏位即可获得永久 Buff，取出后自动移除：

| 可爱标志 | Buff | 台词 |
|---|---|---|
| 苹果嘉儿 | 急迫 II | "我可没学到什么！" |
| 碧琪 | 跳跃提升 II | "又好笑又好紧张！" |
| 柔柔 | 生命恢复 I | "好耶…" |
| 珍奇 | 修补 | "这真是……最！糟！糕！的！事！情！" |
| 云宝 | 速度 II | "大概要再酷 20%" |
| 紫悦 | 水下呼吸 I + 夜视 I | "友谊就是魔法！" |

> 安装 [Curios](https://www.curseforge.com/minecraft/mc-mods/curios) 后解锁「可爱标志」专用饰品栏位，装备后在左腿外侧显示对应贴图。

### 方块

苹果、极地、熊皮、骨头、骨砖、骨矿、骨墙、荧骨、水晶、黑晶、硬苹果、凤凰、机器人、紫悦方块

### 合成配方

91 个配方覆盖方块合成/拆解、材料、武器、投掷物、盔甲、工具。详见 `data/mythicalcreatures/recipes/`。
