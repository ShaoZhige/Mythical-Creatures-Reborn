# Mythical Creatures Reborn

将 MLP Mythical Creatures (1.7.10) 移植到 Minecraft 1.20.1 Forge 并全面革新。


## 模组信息

| | |
|---|---|
| MC 版本 | 1.20.1 |
| 加载器 | Forge 47.2.0+ |
| 许可 | MIT |
| 作者 | Shao_Zhige / fffqu / Holy Light Radiance |

## 内容概览

A modernized 1.20.1 Forge port of the MLP Mythical Creatures mod，159+ 物品、17 个方块、8 套盔甲、9 种实体、102 个合成配方。

> 详细内容见 [Wiki](https://github.com/ShaoZhige/Mythical-Creatures-Reborn/wiki)。

## 联动

| 模组 | 作用 |
|---|---|
| [Curios](https://www.curseforge.com/minecraft/mc-mods/curios)（可选） | 解锁可爱标志饰品栏位 + 腿部渲染 |

## 注意事项

- 需要 **JDK 17** 编译
- 需要安装前置模组GeckoLib
